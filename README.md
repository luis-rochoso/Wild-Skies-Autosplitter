# Wild-Skies-Autosplitter
Livesplit timer autosplitter for Dreamworks Dragons: Wild Skies speedrunning.
Uses the Auto Splitting Language (ASL).

# What does it do:
Wild Skies is a game full of loading screens and their loading times vary wildly depending on the machine running it.
This script automatically pauses the timer when the game is loading, therefore subtracting the load times from the total.
This makes for more accurate timing of the runs and also negates the time discrepancy created by varying loading times.
